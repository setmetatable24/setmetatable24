# MathUtils
Приветствую. Я 16 летний Luau разработчик. Решил опен соурснуть мой модуль MathUtils, этот модуль - строго типизировання математическая библиотека для роблокса. Моя библиотека построена на принципах евклидовой геометрии и линейном алгебре в 3д пространстве. Изначально я планировал сделать этот модуль как набор утилит, но все зашло немного дальше.
В самом скрипте есть расчет расстояний от точки до прямой, отрезка и плоскости, проекции и отражения векторов, ближайшие точки на геометрических примитивах, пересечение лучей, отрезков и плоскостей, билинейная и трилинейная интерполяция, матрица поворота 3x3, кватернионы (Сами кватернионы в этом модуле как инструмент для работы с ориентацией, операции с базисами, решение квадратных и кубических уравнений).
Я думаю это поможет вам решить некоторые проблемы с математикой для игры! В будущем, может быть буду и обновлять этот модуль.
